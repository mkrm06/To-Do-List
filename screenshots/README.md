# Screenshots Directory

This folder contains screenshots of the To-Do List Management System application.

## Required Screenshots

Please add the following screenshots to this folder:

1. **login_window.png**
   - Screenshot of the login/register window
   - Should show the username and password fields, Login and Register buttons

2. **main_window.png**
   - Screenshot of the main application window
   - Should show the task input form on the left and empty task list on the right

3. **task_management.png**
   - Screenshot showing task management features
   - Should display the main window with several tasks added
   - Show the form with task details filled in

4. **task_list.png**
   - Screenshot of the task list with various tasks
   - Should show tasks with different priorities (High/Low)
   - Should show both completed (✓) and pending (○) tasks
   - Should display different categories

## Screenshot Guidelines

- **Format**: PNG or JPG
- **Size**: 800-1200 pixels width recommended
- **File Size**: Keep under 500KB per image
- **Quality**: Clear, readable text and UI elements
- **Naming**: Use exact filenames as listed above

## How to Take Screenshots

### Windows:
- Use `Win + Shift + S` for Snipping Tool
- Or use `Alt + Print Screen` for active window

### macOS:
- Use `Cmd + Shift + 4` for selection screenshot
- Or use `Cmd + Shift + 3` for full screen

### Linux:
- Use `Print Screen` key
- Or use screenshot tools like `gnome-screenshot` or `scrot`

## Notes

- Remove any personal information from screenshots
- Ensure the application window is clearly visible
- Crop screenshots to focus on the application window
- Use consistent window sizes across screenshots
